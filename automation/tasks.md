# Automated Task List
> Scheduled tasks that agents can perform automatically.

## 🕒 Daily Tasks (9:00 PM every day)
1. **Daily log generation**: Automatically organize the day's work content, learning records, problem solving processes, and generate daily log drafts
2. **Resource sorting**: Automatically classify and organize the resources collected that day
3. **Experiment record sorting**: Automatically organize the experimental records generated that day
4. **Commit submission**: Submit the above content to the repository after generation (marked as draft)

## 🕒 Weekly Tasks (8:00 PM every Sunday)
1. **Weekly summary generation**: Automatically summarize the week's work content, learning outcomes, experiment results, and generate weekly report drafts
2. **Content classification**: Automatically classify and organize the week's notes, experiments, and resources
3. **Data statistics**: Count the week's commit quantity, note quantity, experiment quantity and other data
4. **Repository optimization**: Automatically organize repository structure and update index files

## 🕒 Irregular Tasks
1. **Template update**: Regularly update and optimize various document templates according to usage
2. **Content indexing**: Regularly rebuild content indexes to facilitate retrieval
3. **Duplicate cleaning**: Regularly clean up duplicate content and invalid links
4. **Backup**: Regularly back up repository content to prevent data loss

## 📝 Task Execution Specifications
1. All automatically generated content must be marked as draft
2. The commit message must clearly indicate the content type and generation method
3. No more than 5 commits per day for automated operations
4. If an error occurs during task execution, stop immediately and leave a log
