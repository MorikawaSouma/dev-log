## 学习笔记

### Firefox 重入问题

在 Firefox 中，当主线程被 alert/debugger 模态框阻塞时，WebSocket onmessage 事件仍会触发，导致可重入调用 React 工作循环。解决方案是提前退出而非抛出错误。

### TypeScript 类型推断

直接在对象字面量初始化属性比使用 `Object.assign` 更利于 TypeScript 类型推断。

### React Compiler NaN 比较

使用 `Object.is` 语义比较依赖，正确处理 `NaN !== NaN` 的情况。
