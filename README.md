## ⏱️ Update Rhythm

## 🤖 Automation Maintenance Boundary
This repository supports partial automated maintenance by agents, with clear boundaries:
- ✅ **Auto-write allowed**: Daily log drafts, learning notes drafts, resource collection drafts, experiment record drafts, weekly report first drafts
- ⚠️ **Manual confirmation required**: External evaluations, project conclusions, recommended content, judgments involving other people's projects, any content that looks like "public endorsement"
- ❌ **Auto-execution prohibited**: Batch follow/star/watch, automatic comments on other people's projects, automatic issue/PR opening, any behavior of forging reputation or interaction
